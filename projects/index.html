Chitti - Alexa Skill

2016-08-22 :: hARRy

Home Automation is my dream ever since I watched Endhiran Movie in 2010. I’m a big fan of Shankar and I really likes the Character Chitti.

I wanted to create my own version of Chitti which can be my personal assistant. Amazon launching of really helped to achive/live this dream.

Chitti is an Alexa Skill built to handle all my media requests. This skill uses the media library thats build in mongodb, integrates with Plex and Kodi. Handles all audio/video playback requests in various speakers around the house.

Below are some of the Intents that Chitti can handle

  • Play latest/new [language] songs
  • Play [90s/2ks/80s] [language] songs
  • Play composer [A R Rahman/Ilaiyaraja/Harris] songs
  • Play [melodies/party/dance] songs
  • Rate this songs as 4 star
  • What is playing now

Technology Stack - AWS, Alexa SDK, NodeJs, Mongodb, Kodi/Plex/Home Automation APIs
