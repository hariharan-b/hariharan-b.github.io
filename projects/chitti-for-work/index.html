Chitti - DevOps & DW/BI metadata Personal Assistant

2019-02-03 :: hARRy

ACCIO is a front end web application built to handle various project management tasks which was manually maintained in excel sheet.

Manager and team leads maintains multiple version of excel sheet for the projects that team is currently working on. Lot of manual effort to syncup the statues and planning capacity every month.

Technology Stack - MEAN stack web development - Mongodb, Express, AngularJs, NodeJs

Below are some of the major features the tool provides

  • Interactive screen to create projects and keep track of status updates during weekly team meeting
  • One version of truth about all projects and its attributes.
  • Dashboard reports for project counts per release month, Leads, DA and developers
  • Forecast planning for each individual team members and assign exact hours to be logged against each project based on the estimation
  • One click export to excel if reports has to be sent to upper/client management
  • Hierarchial User module to handle login and access restriction to features
